تطبيق ويب تقدمي خفيف وسريع يحتوي على 12 حاسبة صحية ولياقية احترافية
نبذة عن المشروع
حزمة حاسبات اللياقة والصحة هو تطبيق ويب تقدمي (PWA) عالي الأداء وخفيف للغاية، يضم 12 أداة حسابية أساسية ومهمة جدًا في مجال اللياقة البدنية والتغذية والصحة العامة.
تم تطوير المشروع بالكامل بواسطة أخصائي معلوماتية صحية ومطور Full-Stack، بهدف توفير أدوات دقيقة وسريعة تعمل بدون إنترنت وبدون تثبيت معقد.
الحاسبات الموجودة في التطبيق (12 حاسبة احترافية)

مؤشر كتلة الجسم (BMI)
احتياجات السعرات الحرارية اليومية (TDEE & BMR)
نسبة الدهون في الجسم (Body Fat %)
حساب المعدل الأقصى لضربات القلب (Max Heart Rate)
حساب الـ One-Rep Max (1RM) لتمارين القوة
حاسبة السرعة والوتيرة (Pace Calculator) للجري والمشي
حاسبة الـ Macro Nutrients (البروتين، الدهون، الكاربوهيدرات)
حاسبة معدل الأيض الأساسي (BMR) بأكثر من معادلة
حاسبة كمية الماء اليومية
حاسبة نسبة الخصر إلى الورك (Waist-to-Hip Ratio)
حاسبة الحرق أثناء التمرين (Calories Burned)
حاسبة الوزن المثالي (Ideal Body Weight)

الميزات الرئيسية

يعمل 100% بدون إنترنت بعد الزيارة الأولى (Offline-First PWA)
قابل للتثبيت على أندرويد، آيفون، ويندوز، ماك، لينكس
دعم كامل للوحدات المترية والإمبراطورية (كجم/سم ←→ رطل/قدم)
حفظ جميع النتائج محليًا في المتصفح (Local Storage)
تصميم متجاوب وسريع جدًا (أقل من 200 كيلوبايت حجم التطبيق كاملاً)
واجهة أنيقة وبسيطة مع دعم الوضع الليلي (Dark Mode)

التقنيات المستخدمة

HTML5 + CSS3 + Vanilla JavaScript (بدون أي إطار عمل ثقيل)
Progressive Web App (PWA) مع Service Worker
Local Storage لحفظ البيانات

Fitness & Health Calculator Suite – PWA
A lightweight, offline-first Progressive Web App featuring 12 essential fitness and health calculators
Project Overview
This is a fast, ultra-lightweight Progressive Web App that bundles 12 professional-grade fitness and health calculators into a single, installable web application. Built by a Health Informatics Specialist and Full-Stack Developer, it’s designed to work perfectly offline, on any device, with zero dependencies.
Included Calculators (12 Professional Tools)

Body Mass Index (BMI)
Daily Calorie Needs (TDEE + BMR)
Body Fat Percentage
Maximum Heart Rate
One-Rep Max (1RM) Strength Calculator
Running/Walking Pace Calculator
Macro Nutrient Split Calculator
Basal Metabolic Rate (Multiple Formulas)
Daily Water Intake Calculator
Waist-to-Hip Ratio
Calories Burned During Exercise
Ideal Body Weight Calculator

Key Features

100% offline functionality after first visit (true Offline-First PWA)
Installable on Android, iOS, Windows, macOS, Linux
Full Metric & Imperial unit support (kg/cm ↔ lb/ft)
All results saved locally (Local Storage)
Extremely lightweight (< 200 KB total size)
Beautiful, responsive UI with Dark Mode support

Tech Stack

Pure HTML5 + CSS3 + Vanilla JavaScript (no frameworks)
Progressive Web App with Service Worker
Local Storage for data persistence
Fully responsive Mobile-First design
Custom icons + complete web manifest
